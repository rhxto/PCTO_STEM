# Corrente elettrica
è un flusso ordinato di cariche

Intensità di corrente (A) = deltaQ(C)/deltaT(s)

=> conduttore (o = elettrone)

Senso di marcia = da + a -
  ------------------------
+ | o o o o o o o oo o o | -
  ------------------------

=> Unità di misura della corrente = 1 Ampere
Se l'intensità di corrente non cambia è detta "continua"
Misua con tester (aka Multimetro) -> Voltmetro / Amperometro / Reistenza / Altro

Circuito dove la corrente è in grado di passare => chiuso, altrimenti aperto

## Generatore di tensione / potenziale
Alimentatore / pila (Generatore di tensione continua)
Tenesione elettrica = differenza di potenziale

## Def circuito elettrico

Frecce = senso corrente


Serie -> + utilizzatori concatenati (vecchie lampadine di natale)
pila ----X-------X---------X--- pila

Parallelo -> + utilizzatori in parallelo
     /------X-----\
Pila--------X------- Pila
     \------X-----/

=> Multimetro x volt -> serie
=> Multimetro x Ampere -> parallelo

## Resistenza elettrica
R = deltaV/i => 1 Ohm(Ω) -> 1V/1A

=> intensità di corrente = deltaV/R (Ω)
=> Prima legge di Ohm -> la resistenza di corrente è direttamente proporzionale alla differenza di potenziale applicata
ai capi del conduttore
